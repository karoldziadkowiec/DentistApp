# DentistApp34
The project presents a window application with an interface for operating a dental office. The application helps to automate processes in the dental office. The project was created in C++. The application has two forms of use: from the patient's side and from the administrator's (dentist's) side. The functionalities of the application include: registering patients, storing patient files, displaying the next visits, as well as reporting on the state of the warehouse.
